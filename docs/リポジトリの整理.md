# リポジトリの整理

このドキュメントは、**何をコミット・プッシュし、何をローカルに留めるか**を整理するためのメモです。

---

## コミット・プッシュするもの

- **ソースコード**: `src/`（`src/app/`, `src/proxy.ts`）, `scripts/`, `public/`（テンプレート含む）
- **設定**: `package.json`, `tsconfig.json`, `next.config.ts`, `postcss.config.mjs`, `config/eslint.config.mjs`
- **ドキュメント**: `docs/` 配下の Markdown、`docs/samples/`（pdfs / images / sheets）
- **雛形**: `.env.example`（値は空。環境変数名の一覧用）
- **ルート**: `README.md`, `.gitignore`

---

## プッシュしないもの（.gitignore）

| カテゴリ | パターン | 理由 |
|----------|----------|------|
| 環境変数 | `.env*`, `env` | APIキー・シークレットを含むため |
| 依存 | `node_modules/` | `npm install` で再生成 |
| ビルド | `.next/`, `out/`, `build/` | ビルドで再生成 |
| Vercel | `.vercel/` | デプロイ先の紐づけ（環境ごと） |
| ログ | `logs/` | スタンドアロンOCRの出力 |
| IDE | `.claude`, `.cursor`, `*.code-workspace` | ローカル設定 |
| 一時 | `~$*`, `*.tsbuildinfo`, `next-env.d.ts` | 自動生成・ロックファイル |
| その他 | `.DS_Store`, `*.pem`, `venv/`, `__pycache__/` 等 | OS・言語の一時ファイル |

---

## フォルダ全体の役割

```
ocr-main/
├── src/           … コミット（app/, proxy.ts）
├── config/        … コミット（eslint.config.mjs）
├── public/        … コミット（静的・テンプレート）
├── docs/          … コミット（ドキュメント・サンプル）
├── scripts/       … コミット（スタンドアロンスクリプト）
├── next.config.ts, tsconfig.json, postcss.config.mjs, package.json … ルート設定
├── .env.example   … コミット（雛形のみ）
├── .env.local     … プッシュしない
├── env            … プッシュしない
├── node_modules/  … プッシュしない
├── .next/         … プッシュしない
├── .vercel/       … プッシュしない
├── logs/          … プッシュしない
├── .claude/       … プッシュしない
└── .cursor/       … プッシュしない（あれば）
```

新しく「これはリポジトリに入れたくない」というファイルができた場合は、`.gitignore` にパターンを追加してください。
